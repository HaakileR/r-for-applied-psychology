# Lab 12: Capstone Mini-Project

## Goal

Practice one key skill from the open textbook using a synthetic Balkan-style psychology dataset.

## Instructions

1. Open RStudio.
2. Open the project folder.
3. Run `scripts/00_setup.R`.
4. Choose one dataset from `data/raw/`.
5. Complete the tasks below.

## Tasks

1. Import the dataset.
2. Use `glimpse()` and `summary()` to inspect it.
3. Identify two numeric variables and one categorical variable.
4. Create one table.
5. Create one figure.
6. Write a short interpretation in plain language.

## Reflection questions

- What does one row represent?
- Which variable would be a useful outcome variable?
- Which variable would be a useful predictor?
- What should a reader know before interpreting the result?

## Submission

Submit:

- one `.R` script;
- one short results paragraph;
- one figure or table.
