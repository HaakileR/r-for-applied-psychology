# Assignment Rubric

| Criterion | Excellent | Good | Needs improvement |
|---|---|---|---|
| Research question | Clear and psychologically meaningful | Understandable | Vague |
| Data inspection | Correctly describes variables | Mostly correct | Missing or unclear |
| Analysis choice | Appropriate and justified | Appropriate but weak explanation | Inappropriate |
| R code | Reproducible and organized | Runs with minor issues | Does not run |
| Interpretation | Clear and cautious | Mostly clear | Overclaims |
| Reporting | Transparent and structured | Adequate | Incomplete |
